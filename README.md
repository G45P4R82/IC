# Acoplamento de Modelos de ML como Componentes de Software
Este repositório contém os artefatos e dados extraídos para o estudo MSR (Mining Software Repositories) da disciplina MC926B/MO630A.
## Estrutura
- `atividade.md`: Protocolo de pesquisa com GQM, RQs e critérios de inclusão.
- `amostra_repositorios_ml.csv`: Dataframe contendo a extração inicial de métricas de acoplamento de repositórios reais no GitHub.
